# Lesson 01 - First GET Request

## Objective

Learn how to send a GET request using Postman and understand the response returned by an API.

## API Used

https://jsonplaceholder.typicode.com/users

## HTTP Method

GET

## What is a GET Request?

A GET request is used to retrieve data from a server. It does not modify or create any data and is considered a safe and idempotent HTTP method.

## Purpose of this Request

Retrieve a list of users from the server.

## Response

The API returns a JSON array containing user information such as:

* id
* name
* username
* email
* address
* phone

## Status Code

200 OK

Meaning:

* 200 indicates that the request was successful.
* The server processed the request and returned the expected data.

## My Learning

* Postman can be used to send HTTP requests to APIs.
* GET requests are used to retrieve data.
* API responses are commonly returned in JSON format.
* Status codes help determine whether a request was successful or failed.
