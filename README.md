# Postman Learning Workspace

A comprehensive API Testing and Postman learning repository.

## Objectives

* Learn API fundamentals
* Master Postman features
* Build reusable collections
* Write API test scripts
* Manage environments and variables
* Execute collections using Newman
* Integrate API testing with GitHub Actions
* Create a professional QA/API Testing portfolio

## Repository Structure

### Lessons

Contains theory notes, examples, and interview questions.

### Collections

Contains exported Postman collections.

### Environments

Contains exported environment files.

### Screenshots

Contains screenshots used in documentation.

### Projects

Contains end-to-end API testing projects.

## Learning Roadmap

1. API Fundamentals
2. HTTP Methods
3. Request Components
4. Collections
5. Variables
6. Environments
7. Authentication
8. Tests and Assertions
9. Collection Runner
10. Newman
11. GitHub Actions
